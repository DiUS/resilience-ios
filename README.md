resilience-ios
==============

Installation
------------
````
gem install xcoder
gem install cocoapods
cd Resilience
pod install
````
Building
--------

### To build and run unit tests:
````
rake debug:build
````
### To upload to testflight:
````
rake debug:testflight
````
XCode / AppCode
---------------
open the Resilience workspace: `Resilience.xcworkspace`
